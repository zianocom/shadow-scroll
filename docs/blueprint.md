# **App Name**: Shadow Scroll

## Core Features:

- Featured Article: Display a hero section with a captivating featured mystery article on the landing page.
- Latest Posts: Showcase a list of the most recent mystery posts with 'Read More' buttons for easy navigation.
- Detailed Article Page: Create individual pages for each mystery article with immersive design and clear typography.
- SEO Optimization: Implement SEO-optimized meta tags for improved search engine visibility.

## Style Guidelines:

- Dark color scheme with a mysterious, cinematic feel. Background color: Deep charcoal (#222222).
- Primary color: Intense, desaturated red (#B33F40), reminiscent of old film noir.
- Accent color: Deep amber (#C9772E) for highlighting important elements like 'Read More' buttons and links.
- Font pairing: 'Playfair' (serif) for headlines and titles, providing an elegant, high-end feel. 'PT Sans' (sans-serif) for body text.
- Cinematic layout with full-width sections and strategically placed imagery to enhance the immersive experience.
- Subtle animations and transitions to create a smooth and engaging reading experience.
